# Neon Grid Quest

A premium grid-based puzzle game interface built with React, Tailwind CSS, and Framer Motion.

## Features

- Responsive grid generation from `10x10` up to `30x30`
- Three interactive tile types: blue points, red danger, and green safe zones
- Pattern-based grid generation with two difficulty modes
- Animated HUD, win/loss overlay, and smooth transitions
- Clean, modular component structure ready for deployment

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start development server:

   ```bash
   npm run dev
   ```

3. Build for production:

   ```bash
   npm run build
   ```

## Deploy

This project can be deployed to Vercel or Netlify from the repository root.
